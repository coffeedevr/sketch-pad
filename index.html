Sketch Pad

Color Picker: #000000

Paint Controls:

Brush

Eraser

Utilities:

Grid Display

Erase

Canvas Size: 12x12

Instructions

To start drawing, use the Brush tool.

If you wish to erase a color, use the Eraser tool.

You can toggle canvas borders by clicking the Grid Display Button.

You can reset the canvas by clicking the Erase button.

Tips!

You can hold the button while clicking on a grid to add tint!

You can hold the button while clicking on a grid to add shading!
